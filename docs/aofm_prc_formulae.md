03 Pricing Formulae

Similar to most fixed income securities in Australia, Treasury Bonds are quoted and traded on a yield to maturity basis rather than on a price basis. This means the price is calculated by inputting the yield into the appropriate pricing formula.

The price per $100 face value is calculated using the following pricing formulae:

(1) Basic formula

$$P = v _ { a } ^ { \frac { f } { d } } \left( g ( 1 + a _ { n } ) + 1 0 0 v ^ { n } \right)$$

(2) Ex-interest formula

$$P = v _ { \overline { { n } } } ^ { \overline { { f } } } \left( g a _ { \overline { { n } } } + 1 0 0 v ^ { \overline { { f } } } \right)$$

The ex-interest period for Treasury Bonds is seven calendar days. With ex-interest Treasury Bonds the next coupon payment is not payable to a purchaser of the bonds. In this case, calculation of an ex-interest price is effected by the removal of the '1' from the term:

$$1 + a _ { n }$$

in formula (1), thereby adjusting for the fact that the purchaser will not receive a coupon payment at the next interest payment date.

(3) Near-maturing bonds formula (between the record date for the second last coupon and the record date for the final coupon)

$$P={\frac {100+g}{1+\left({\frac {f}{f_{s}}} \right)i}}$$

When a Treasury Bond goes ex-interest for the second last time it is treated as a special case. In this case formula (3) applies up until the record date for the final interest

payment. There may be a slight discontinuity in the progress of the price of the bond around the time the bond goes ex-interest for the second last time but market

participants can, if they wish, allow for this in their trading.

Where the maturity date coincides with a weekend or public holiday, the commonly accepted practice is to price near-maturing Treasury Bonds according to the actual date the principal and final interest are paid (and not the nominal maturity date).

## (4) Near-maturing bonds formula (between the record date for the final coupon and maturity of the bond)

$$P = { \frac { 1 0 0 } { 1 + \left( { \frac { f } { 3 6 5 } } \right) i } }$$

When a Treasury Bond goes ex-interest for the last time it is treated as a special case. In this case formula (4) applies from the time the bond goes ex-interest for the final time.

Where the maturity date coincides with a weekend or public holiday, the commonly accepted practice is to price near-maturing Treasury Bonds according to the actual date the principal and final interest are paid (and not the nominal maturity date).

### In these formulae:

$P=$ the price per $100 face value. $P$ is rounded to three decimal places in formulae (1) and (2), and unrounded in formulae (3) and (4) $i=$ the annual percentage yield to maturity divided by 200 in formulae (1) and (2), or the annual percentage yield to maturity divided by and (4), if the maturity date falls on a non-business day, the next good business day (as defined in the Information Memorandum) is used $d=$ the number of days in the half year ending on the next interest payment date. $g=$ the half-yearly rate of coupon payment per $100 face value. $n=$ the term in half years from the next interest payment date to maturity. $a_{n1}=v+v^{2}+...+v^{n}={\frac {1-v^{n}}{i}}$. Except if $i=0$ then $a_{n1}=n$

## Worked Examples

### (1) Basic formula

Consider the 2.75% 21 November 2029 Treasury Bond, with a yield to maturity of 1.10 per cent and settlement date of 12 September 2019.

$$P = v _ { a } ^ { L } \left( g ( 1 + a _ { n } ) + 1 0 0 v ^ { n } \right)$$

$$ where: $$

$$i = { \frac { 1 . 1 0 } { 2 0 0 } } = 0 . 0 0 5 5$$

$$v = { \frac { 1 } { 1 + i } } = { \frac { 1 } { 1 + 0 . 0 0 5 5 } } = 0 . 9 9 4 5 3$$

$$a _ { \overline { { n } } } = { \frac { 1 - v ^ { n } } { i } } = { \frac { 1 - 0 . 9 9 4 5 3 ^ { 2 0 } } { 0 . 0 0 5 5 } } = 1 8 . 8 9 0 4$$

$f=70$, the number of days from 12 September 2019 to 21

$d=184$, the number of days from 21 May 2019 to 21 M

$$g = { \frac { 2 . 7 5 } { 2 } } = - 1 . 3 7 5$$

n = 20, the number of half years from 21 November 2019 to 21 M

$$P = v _ { a } ^ { I } \left( g ( 1 + a _ { n } ) + 1 0 0 v ^ { n } \right) = 0 . 9 9 4 5 3 ^ { \frac { 7 0 } { 1 8 1 } } \left( 1 . 3 7 5 \left( 1 + 1 8 . 8 9 0 4 \right) + 1 0 0 \times 1 0 ^ { - 2 } \right) .$$

$$P = 1 1 6 . 7 1 6$$

### (2) Ex-Interest formula

Consider the 2.50% 21 May 2030 Treasury Bond, with a yield to maturity of 1.10 per cent and settlement date of 15 November 2019.

$$P=v^{\frac{d}{d}}\left(g\left(a_{\overline{m}}\right)+100 v^{n}\right)$$

where: $i={\frac {1.10}{200}}=0.0055$ $v={\frac {1}{1+i}}={\frac {1}{1+0.0055}}=0.99453$ $a_{n1}={\frac {1-v^{n}}{i}}={\frac {1-0.99453^{21}}{0.0055}}=$19.78135 $f=6$, the number of days from 15 November 2019 to 21 November 2019 $d=184$, the number of days from 21 May 2019 to 21 November 2019 $g={\frac {2.50}{2}}=1.25$ $n=21$, the number of half years from 21 November 2019 to 21 May 2030

$$P=v^{\frac{f}{d}}\left(g\left(a_{\overline{n}}\right)+100 v^{n}\right)=0.99453^{\frac{6}{184}}\left(1.25(19.78135)+100 \times 0.99453^{21}\right)$$

$$P = 1 1 3 . 8 2 7$$

### (3) Near-maturing bonds formula (between the record date for the second last coupon and the record date for the final coupon)

Consider the 2.75% 21 October 2019 Treasury Bond, with a yield to maturity of 1.00 per cent and settlement date of 26 September 2019.

$$P={\frac {100+g}{1+{\Bigl (}}{\tfrac {f}{365}}{\Bigr )}i$$

where: $i={\frac {1.00}{100}}=0.01$ $f=25$, the number of days from 26 September 2019 to 21 October 2019 $g={\frac {2.75}{2}}=1.375$ $P={\frac {100+g}{1+\left({\frac {f}{365}}\right)i}}={\frac {100+1.375}{1+\left({\frac {25}{365}}\right)\times 0.01}}$ $P=101.305613$
